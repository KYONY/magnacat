## Full Description (English)

Magnacat Translator — a smart Chrome extension for instant translation between Ukrainian and English using Google Gemini AI.

FEATURES:

• Instant Translation — Select any text on a webpage and get translation in a tooltip
• Text-to-Speech — Listen to the original text with natural AI voice
• Text Replacement — Replace selected text in input fields with translation
• Keyboard Shortcut — Quick translate with Ctrl+Shift+B (customizable)
• Context Menu — Right-click to translate selected text
• Multiple Models — Choose from various Gemini models for translation and TTS
• Theme Support — Light, Dark, or System theme

HOW TO USE:

1. Get your free Gemini API key at ai.google.dev
2. Enter the API key in extension settings
3. Select text on any webpage
4. Click the translation icon or use keyboard shortcut

Perfect for:
• Learning Ukrainian or English
• Reading foreign articles and documentation
• Quick translations while browsing

Privacy: Your API key is stored locally. Text is sent to Google Gemini API for translation only.
