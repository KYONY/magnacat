## Full Description (English)

Magnacat Translator — a smart Chrome extension for instant translation between Ukrainian and English using Google Gemini AI.

FEATURES:

• Instant Translation — Select any text on a webpage and get translation in a tooltip
• Text-to-Speech — Listen to the original text with natural AI voice
• Text Replacement — Replace selected text in input fields with translation
• Keyboard Shortcut — Quick translate with Ctrl+Shift+B (customizable)
• Context Menu — Right-click to translate selected text
• Multiple Models — Choose from various Gemini models for translation and TTS
• YouTube Subtitle Translation — Click any word in YouTube subtitles to translate; Shift+click to select multiple words; video auto-pauses during translation
• Theme Support — Light, Dark, or System theme

HOW TO USE:

1. Get your free Gemini API key at ai.google.dev
2. Enter the API key in extension settings
3. Select text on any webpage
4. Click the translation icon or use keyboard shortcut

YouTube subtitles:
1. Enable "YouTube subtitles" in settings
2. Open a YouTube video with captions on
3. Click any word in subtitles — video pauses, translation appears
4. Close tooltip to resume playback

Perfect for:
• Learning Ukrainian or English
• Reading foreign articles and documentation
• Quick translations while browsing
• Learning languages with YouTube videos

Privacy: Your API key is stored locally. Text is sent to Google Gemini API for translation only.

## Full Description (Ukrainian)

Magnacat Translator — розумне розширення Chrome для миттєвого перекладу між українською та англійською за допомогою Google Gemini AI.

МОЖЛИВОСТІ:

• Миттєвий переклад — Виділіть будь-який текст на сторінці та отримайте переклад у тултіпі
• Озвучування тексту — Прослухайте оригінальний текст природним голосом AI
• Заміна тексту — Замініть виділений текст у полях введення на переклад
• Гарячі клавіші — Швидкий переклад через Ctrl+Shift+B (можна змінити)
• Контекстне меню — Клік правою кнопкою для перекладу виділеного тексту
• Вибір моделей — Оберіть з різних моделей Gemini для перекладу та озвучки
• Переклад субтитрів YouTube — Клікніть на будь-яке слово в субтитрах YouTube для перекладу; Shift+клік для вибору кількох слів; відео автоматично призупиняється
• Підтримка тем — Світла, темна або системна тема

ЯК КОРИСТУВАТИСЯ:

1. Отримайте безкоштовний Gemini API ключ на ai.google.dev
2. Введіть API ключ у налаштуваннях розширення
3. Виділіть текст на будь-якій сторінці
4. Натисніть іконку перекладу або використайте гарячі клавіші

Субтитри YouTube:
1. Увімкніть "YouTube subtitles" у налаштуваннях
2. Відкрийте відео на YouTube з увімкненими субтитрами
3. Клікніть на слово в субтитрах — відео зупиниться, з'явиться переклад
4. Закрийте тултіп для продовження відтворення

Ідеально для:
• Вивчення української або англійської мови
• Читання іноземних статей та документації
• Швидких перекладів під час серфінгу
• Вивчення мов за допомогою відео на YouTube

Конфіденційність: Ваш API ключ зберігається локально. Текст надсилається до Google Gemini API лише для перекладу.
